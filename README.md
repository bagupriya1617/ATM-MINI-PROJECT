📌 Overview
-->SimpleATM is a beginner-friendly C++ console application that simulates basic ATM operations.
-->It allows users to check balance, deposit money, and withdraw money with simple menu-driven interaction.

✨ Features
1. Check account balance
2. Deposit money
3. Withdraw money with validation (no overdraft)
4. Menu-driven interface
5. Beginner-friendly C++ code structure

🛠️ Technologies Used
Language: C++
Concepts: Classes, Functions, Loops, Conditionals
